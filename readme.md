# NEDES - Example GitHub Website (Static)

This repository contains a small static website (3 files) that links to five NEDES blog posts (from the sitemap you provided). It's ready to be uploaded to a GitHub repository (or GitHub Pages).

## Included files
- `index.html` — the website landing page with links to selected NEDES articles.
- `style.css` — simple stylesheet used by `index.html`.
- `README.md` — this file.

## Linked NEDES blog posts
The site links to these posts (as requested):
- https://nedes.us/solar-removal-and-installation-services-a-complete-overview/
- https://nedes.us/the-ultimate-guide-for-residential-solar-installation/
- https://nedes.us/how-much-do-solar-panels-cost/
- https://nedes.us/solar-panel-maintenance-tips/
- https://nedes.us/residential-solar-installation-cost-2025/

## How to use
1. Download the three files in this folder.
2. Create a new GitHub repository (public or private).
3. Upload `index.html`, `style.css`, and `README.md` to the repo root.
4. (Optional) Enable GitHub Pages from repository Settings → Pages to publish the site.
